FEATURES

Every feature you need to manage
OpenClaw agents

30+ features built by an OpenClaw user who got tired of SSH-ing into Railway to check if things were working.

Live Agent Monitoring

See your OpenClaw agent's status in real time — heartbeat, latency, active sessions, and connection health. Get instant alerts when something goes wrong. No more SSH-ing into Railway or checking server logs.

Auto-Throttle

Set daily and monthly spend limits on your OpenRouter API usage. When your spend approaches the limit, Mission Control automatically switches your OpenClaw agent to cheaper models like DeepSeek V3 ($0.20/MTok) instead of Claude Sonnet ($3/MTok). Users save an average of $48 per week.

Pipeline Tracking

Follow every pipeline job step-by-step as it moves through Scout, Editor, Narrator, and Assembly stages. See per-step cost breakdown, execution time, and model usage. Catch bottlenecks before they become expensive problems.

One-Click Mode Switch

Switch between Performance, Standard, and Budget modes with a single click. Performance mode uses top-tier models for quality-critical tasks. Budget mode cuts costs by 85% for routine work. No SSH, no config files, no terminal — just click.

Cost Analytics

Track spending by model, by day, by pipeline step, and by channel. Import your OpenRouter CSV billing data for historical analysis. Visualize cost trends, identify the most expensive operations, and see exactly where every dollar goes.

Multi-Channel View

See all your agent's activity across every channel in one unified view — Discord servers, direct messages, automated pipelines, and scheduled tasks. Filter by channel, search by keyword, and never miss what your agent is doing.

Multi-Provider Cost Comparison

Compare what your workload costs across 12 models from Anthropic, OpenAI, Google, Deepseek, and local providers. Ranked by price with potential savings calculated from your actual usage data.

AI Usage Forecasting

See projected end-of-month spending with a visual budget gauge, days-until-budget-exceeded, and smart recommendations like "switch to Gemini Flash to save 96%" based on your real usage patterns.

Session Replay

Browse every agent conversation with full message history, per-message token costs, model attribution, and full-text search. Understand exactly where your tokens went and what your agent said.

Smart Alerts + Webhooks

Configurable alert rules with OS notifications, plus webhook integrations that fire to Slack, Discord, or any HTTP endpoint when thresholds are crossed.

Model Presets

One-click agent configurations: Fast + Cheap, Balanced, Maximum Quality, Code Expert, Long Context, or create your own custom presets.

Backup, Snapshots + More

Full data backup/restore, config snapshots, prompt library, API key vault, cost tagging, keyboard shortcuts, audit log, guided onboarding, and 3 themes with customizable accent colors.

WHY MISSION CONTROL

Built different

Unlike cloud-based observability tools, Mission Control is a local desktop app. Your data never leaves your machine.

Cloud dashboards

Tools like Langfuse and Helicone are great for observability, but they're cloud-hosted, require integration code, and charge based on usage volume. They also don't manage your agent — they only watch it.

Raw OpenRouter dashboard

OpenRouter shows your credits and usage, but it can't switch your agent's model, set budget alerts, track pipeline jobs, or auto-throttle when costs spike. It's a billing page, not a control panel.

Mission Control

A local desktop app that both monitors and controls your OpenClaw agent. Set spend limits, auto-throttle to budget models, track pipeline jobs, switch modes — all without SSH or sending data to a third party.
