How to install

1

Download the installer

Choose the installer for your platform above. The download starts immediately.

2

Run the installer

Double-click the file on Windows or Mac. On Linux, run chmod +x on the AppImage first.

3

Launch & set up

Open Mission Control and follow the setup wizard to connect your OpenClaw instance.

These downloads are installable builds. Activate Mission Control with a license key from the pricing page after installation.
