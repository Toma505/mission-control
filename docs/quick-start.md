# Mission Control Quick Start

## What You Need

- A Mission Control license key
- A reachable OpenClaw instance
- Your OpenClaw URL
- Your OpenClaw setup password

## First Run

1. Install Mission Control for your platform.
2. Launch the app.
3. Enter your license key on the activation screen.
4. Continue to setup.
5. Enter your OpenClaw URL and setup password.
6. Run the connection test.
7. Save the connection.
8. Open the dashboard.

## Recommended First Configuration

- Set a daily and monthly budget in `Costs`
- Choose a default mode in `Costs`
- Open `Preferences` and set your theme, accent, and desktop behavior
- Open `About & Diagnostics` once so you know where logs and app data live

## First Things To Check

- Dashboard loads live data
- Mode switching works
- Budget save works
- Notifications appear
- Workflow Builder opens and saves changes

## Need Help

- Invalid key or activation issue: check the exact key you received after checkout
- Connection error: verify the OpenClaw URL and setup password
- Recovery state: open `About & Diagnostics`, copy the report, and include it in support requests
