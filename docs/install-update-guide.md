# Install and Update Guide

## Windows

1. Run the `.exe` installer.
2. Complete the install wizard.
3. Launch Mission Control from the desktop or Start menu.

## macOS

1. Open the `.dmg`.
2. Drag Mission Control to `Applications`.
3. Launch the app from `Applications`.

## Linux

### AppImage

1. Download the AppImage.
2. Mark it executable.
3. Run it directly.

### Debian or Ubuntu

1. Install the `.deb` package.
2. Launch Mission Control from the application menu.

## Updating

- When a public tagged release exists, the desktop app can check for updates from the app menu
- If no public release has been published yet, the updater will say so explicitly
- You can always install the latest build manually from the release page

## Uninstalling

- Remove the app through your platform's normal uninstall flow
- If you also want to remove local app data, delete the Mission Control app data folder after uninstall
